Pengaturan Pengecekan Setting

Admin mengisi nilai setting yang seharusnya, bisa manual atau update massal memakai file Excel hasil export relai.

Format import Excel nilai acuan

Gunakan file Excel dengan format yang sama seperti hasil export setting relai NR PCS-931S. Sistem akan membaca semua sheet yang memiliki header Setting Title dan kolom Value / Value(1) sampai Value(10). Nilai pada kolom tersebut akan disimpan sebagai nilai acuan.

System_SettingsDiff_SettingsDistProt_SettingsOC_SettingsSetting TitleValueValue(1)UnitRange

Import ulang Excel akan menyinkronkan acuan penuh untuk merk dan tipe relai aktif: nilai acuan dan urutan diperbarui, kategori dan toleransi lama tetap dipertahankan, parameter baru ditambahkan, serta parameter lama yang tidak ada di file Excel baru akan dihapus.

Kategori Setting yang Dibandingkan

Pengaturan ini digunakan pada menu Pengecekan Setting. Parameter dengan kategori Nilai Bebas tidak akan dibandingkan.

Mode Penyimpanan Server Supabase

Mode server Supabase sudah ditanam pada file HTML sehingga tidak perlu diatur di setiap komputer. Login memakai Supabase Auth, authority dibaca dari tabel profiles, dan data acuan tersinkron dari tabel Supabase.

Pengaturan User Login Supabase

Daftar user tidak diatur di web. Buat user melalui Supabase Dashboard > Authentication > Users, lalu isi authority pada tabel profiles dengan nilai Admin atau Level 1. Dengan cara ini password dikelola oleh Supabase Auth dan tidak disimpan di HTML/localStorage.

Pengaturan Database Hasil Pengecekan

Isi link folder Google Drive atau link Web App database untuk penyimpanan hasil pengecekan. Pada prototipe standalone, tombol Simpan ke Database akan membuat file Excel dan membuka link ini.

Catatan koneksi Google Sheets

Untuk prototipe HTML ini, data diambil langsung oleh browser melalui format CSV Google Sheets. Pastikan link spreadsheet bisa diakses dan nama sheet/rentang cell benar. Contoh rentang: A2:A untuk satu kolom daftar pilihan.
